# Warning Message

Generates a warning message.

## Usage

``` r
ps_warning(..., call. = FALSE)
```

## Arguments

- ...:

  *either* zero or more objects which can be coerced to character (and
  which are pasted together with no separator) *or* a single
  [condition](https://rdrr.io/r/base/conditions.html) object.

- call.:

  logical, indicating if the call should become part of the warning
  message.
