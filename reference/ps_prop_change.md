# Proportional Change

Proportional Change

## Usage

``` r
ps_prop_change(x, y)
```

## Arguments

- x:

  A vector of the before values

- y:

  A vector of the after values

## Value

A vector of the proportional change

## Examples

``` r
ps_prop_change(3, c(3, 1, 9))
#> [1]  0.0000000 -0.6666667  2.0000000
```
